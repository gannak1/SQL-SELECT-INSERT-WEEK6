## 읽는 순서 추천
1. 과제 범위를 먼저 확인한다.
2. MiniSQL 문장이 어떤 구조로 이루어지는지 이해한다.
3. 입력 -> 파싱 -> 실행 -> 저장 흐름을 본다.
4. 어떤 파일 포맷을 택할지 비교한다.
5. 하드코딩된 테이블 구조를 확인한다.
6. 실제 파일 구조와 함수 책임을 확인한다.
7. MiniSQL 입력 규칙을 확정한다.
8. 지원/미지원 문법 경계를 확인한다.
9. 최종 구현 순서를 정리한다.
